# rhythm_game

チューニズムとシューティングゲームの融合

周りから弾が飛んできて、中央に来たらコントローラを任意の方向に移動させる。
すると球は移動した方向に飛んでいく。
弾がブロックにあたれば、そのブロックは消滅する。
大きな円が現れ、徐々に小さくなるアニメーションが流れたら、中央の円と重なった時にコントローラから手を離す。
すると周りのブロックがランダムで消滅する。
タイミングを逃すとランダムでブロックが１個増える。
